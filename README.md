# Application de Gestion de Projets

Une application complète de gestion de projets avec tâches, développée avec Node.js/MongoDB (backend) et Angular (frontend).

# Fonctionnalités

- **Gestion des projets** (CRUD complet)
- **Gestion des tâches** (CRUD complet)
- **Relation OneToMany** entre projets et tâches
- **Interface Angular** réactive et moderne
- **Validation des données** côté backend et frontend

##  Technologies Utilisées

### Backend
- Node.js
- Express.js
- MongoDB avec Mongoose
- CORS

### Frontend
- Angular 16
- TypeScript
- Bootstrap 5
- Reactive Forms

##  Installation
- npm i
- npm run dev
- Variables d'environnement:.env
- MONGODB_URI=mongodb://127.0.0.1:27017/projet_management
- PORT=3000
